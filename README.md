# 1 Case
Een huisartsenpraktijk wil een simpel afspraken programma. Hiervoor moeten ze afspraken kunnen aanmaken. Daarbij moet, naast het moment, een keuze worden gemaakt welke patiënt een afspraak heeft en welke arts deze afhandelt. Artsen, patiënten en afspraken moeten hiervoor gestructureerd worden bijgehouden. Ze willen uiteindelijk een lijst van al hun afspraken inzichtelijk hebben. 

Dit moet gebouwd worden aan de hand van de gegeven template, geef de functies van Program.cs inhoud hiervoor. 
Voor de arts moeten naam en adres worden bijgehouden, voor de patient ook, maar hier moet ook een contactgegeven worden bijgehouden. 

Probeer niet langer dan 3 uur aan de opdracht te besteden. 


## 1.2 Requirements
- Gebruik maken van de template 
- Maak gebruik van Entity Framework voor opslag 
- Alles moet gaan via de console 
- De source code dient leesbaar en onderhoudbaar te zijn 
- Als de gebruiker foutieve data invult moet hier melding van worden gemaakt 
- Voorzie een unittest


## 1.3 Optioneel
- Maak een extra optie na het kiezen voor toon afspraken om het te tonen voor een specifieke patiënt of een specifieke arts
